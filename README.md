## Notes :smile:
- `#` === `<h1>`
- `##` === `<h2>`
- `(text)[address]` === `<a href="address">text</a>`
- "* *" or "_ _" === Italic
- "** **' or "_ _" === Bold
- "*** ***" === All Bold and Italic
- \```js === javascript
- \```html === html
